# XFCE 移动端开发桌面环境配置

## 桌面环境特性
- **轻量级** - XFCE占用资源少，适合开发环境
- **高度可定制** - 可以针对移动端开发进行优化
- **多显示器支持** - 适合多屏开发工作流
- **终端集成** - 内置优秀的终端模拟器

## 开发者专用配置

### 1. 工作区设置
- **4个工作区**：分别用于不同开发任务
  - 工作区1: Android Studio / IntelliJ IDEA
  - 工作区2: VS Code / Flutter开发
  - 工作区3: 终端和命令行工具
  - 工作区4: 浏览器和文档

### 2. 面板配置
- **顶部面板**：系统状态、时间、网络
- **底部面板**：应用启动器、任务管理器
- **快速启动**：Android Studio、VS Code、Terminal、Flutter DevTools

### 3. 快捷键配置
- `Ctrl+Alt+T` - 打开终端
- `Ctrl+Alt+A` - 打开Android Studio
- `Ctrl+Alt+V` - 打开VS Code
- `Ctrl+Alt+F` - 打开Flutter DevTools
- `Super+1-4` - 切换工作区

### 4. 主题和外观
- **主题**：Adapta-Nokto (深色主题，保护眼睛)
- **图标**：Papirus (现代化图标集)
- **字体**：Fira Code (编程友好字体)

### 5. 开发工具集成
- **终端别名**：
  - `android` - 启动Android Studio
  - `flutter-dev` - 启动Flutter开发环境
  - `kmp` - 进入KMP项目目录
  - `mobile` - 快速访问移动端开发目录

### 6. 自动化脚本
- **启动脚本**：自动配置开发环境
- **备份脚本**：定期备份项目和配置
- **更新脚本**：一键更新所有开发工具

## 安装后的配置步骤

1. **首次登录**：选择XFCE会话
2. **运行配置脚本**：`./configure-mobile-dev-desktop.sh`
3. **安装开发工具**：Android Studio, VS Code等
4. **配置终端**：设置别名和主题
5. **优化性能**：调整系统设置以获得最佳开发体验

## 性能优化建议

- **禁用不必要的视觉效果**
- **增加交换空间**（如果内存不足）
- **配置SSD优化**（如果使用SSD）
- **调整电源管理**（插电时使用高性能模式）

## 故障排除

- **黑屏问题**：检查显卡驱动
- **输入法问题**：配置Fcitx5输入法框架
- **分辨率问题**：手动设置显示器分辨率
- **声音问题**：检查PulseAudio配置
