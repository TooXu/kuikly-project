## 🛠️ 安装方法

### 方法 1: ClawHub CLI（推荐）
```bash
# 安装单个技能
npx clawhub@latest install <skill-slug>

# 示例
npx clawhub@latest install browser-automation
npx clawhub@latest install summarize
npx clawhub@latest install ai-news-oracle
```

### 方法 2: 手动安装
```bash
# 克隆技能仓库
git clone <skill-repo-url> ~/.openclaw/skills/<skill-name>

# 示例
git clone https://github.com/openclaw/skills/tree/main/skills/thesethrose/agent-browser \
  ~/.openclaw/skills/agent-browser
```

### 方法 3: 自动安装（推荐）
```bash
# 直接在对话中粘贴 GitHub 链接
# 例如：https://github.com/openclaw/skills/tree/main/skills/swimmingkiim/ai-news-oracle
# 然后说："安装这个技能"
```

## 💡 使用建议

### 1. 渐进式安装
不要一次性安装太多技能，建议：
- 每周安装 2-3 个
- 每个技能使用 1-2 天后评估
- 记录哪些技能真正有用

### 2. 技能组合
有些技能可以组合使用：
- `browser-automation` + `summarize` = 自动浏览并总结网页
- `ai-news-oracle` + `self-improving-agent` = 新闻学习循环
- `github` + `agent-audit` = 代码质量监控

### 3. 定期清理
- 每月审查已安装技能
- 删除不常用或重复的技能
- 更新到最新版本
