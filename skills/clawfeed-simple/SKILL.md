# ClawFeed Simple

Simplified version of ClawFeed without native SQLite dependency. Uses file-based storage and pure JavaScript.

## Features
- 📰 Basic news digest generation
- 📡 RSS feed aggregation  
- 🌐 Multi-language support
- 🖥️ Web dashboard (static files)

## Setup
```bash
# Install dependencies
npm install

# Start server
npm start
```

## Dependencies
- No native modules required
- Pure JavaScript implementation
- File-based storage instead of SQLite
