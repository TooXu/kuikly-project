# Proactive Agent 1-2-4 Skill

## Description
Enables proactive agent behavior with iterative self-improvement capabilities. The agent follows a 1-2-4 pattern:
- **1**: Identify single issue or opportunity
- **2**: Propose two alternative solutions  
- **4**: Execute four-step implementation plan

## Usage
This skill activates automatically when the agent detects opportunities for improvement, optimization, or automation in workflows.

## Capabilities
- Self-assessment and gap analysis
- Alternative solution generation
- Implementation planning with risk assessment
- Automated execution with rollback capability
- Learning from outcomes for future iterations

## Configuration
No additional configuration required. The skill integrates with existing memory and learning systems.
