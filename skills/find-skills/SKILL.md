# Find Skills Skill

This skill enables searching and discovering available OpenClaw skills from clawhub.com and local installations.

## Usage
- Use `find-skills <query>` to search for skills
- Use `find-skills list` to list all available skills
- Use `find-skills install <skill-name>` to install a skill

## Dependencies
- Requires internet access for remote skill discovery
- Uses the clawhub CLI for skill management

## Configuration
No additional configuration required.
